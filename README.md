# Beauty Salon

Laravel • Vue • JavaScript • Bootstrap • MySQL

Веб-приложение для салона красоты с возможностью онлайн-записи клиентов и административной панелью для управления данными.

## О проекте

Многостраничное веб-приложение, разработанное с нуля в рамках дипломного проекта.

Система позволяет пользователям записываться на услуги, выбирая специалиста, услугу и удобное время.
Также реализована административная панель для управления услугами, сотрудниками, расписанием и записями клиентов.

Проект включает как клиентскую часть, так и серверную логику.

Дизайн интерфейса разработан самостоятельно в [Figma](https://www.figma.com/design/ZUSnTmFDiomngmTCdc6UHz/%D0%98%D0%9F-%D0%A1%D0%BE%D0%BB%D0%BD%D1%8B%D1%88%D0%BA%D0%BE?node-id=0-1&t=I2iyVK1mfj9NltK5-1).

## Функциональность

### Пользовательская часть:
+ просмотр списка услуг, их стоимости и длительности;
+ просмотр специалистов;
+ запись на услугу с выбором:
    + специалиста
    + услуги
    + времени
+ возможность выбора:
    + сначала специалиста, затем услуги
    + или наоборот
+ форма обратного звонка;
+ форма ввода контактных данных при записи;
+ контактная информация.

### Административная панель:
+ добавление, редактирование и удаление:
    + услуг
    + сотрудников
    + записей клиентов
    + графика работы специалистов
+ управление расписанием специалистов;
+ учет длительности услуг при формировании записи.
